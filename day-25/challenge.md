# Day 25: Build a Simple Calculator

## Problem
Create a calculator object with methods for basic arithmetic operations: add, subtract, multiply, and divide.

## Examples
```javascript
calculator.add(5, 3)        // Returns: 8
calculator.subtract(10, 4)  // Returns: 6
calculator.multiply(6, 7)   // Returns: 42
calculator.divide(20, 4)    // Returns: 5
calculator.divide(10, 0)    // Returns: "Error: Division by zero"
```

## Bonus
Add chaining support:
```javascript
calculator.add(5, 3).multiply(2).subtract(4)  // Returns: 12
```
